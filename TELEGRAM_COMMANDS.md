# Telegram Channel Performance Updates

## Quick Commands

### Send Current Performance Update
Sends ROI, profit, and win rate stats to your channel:
```bash
python3 send_performance_update.py
```

### Send Weekly Performance Report
Sends detailed weekly report with score breakdown:
```bash
python3 weekly_performance_report.py
```

## What Gets Posted Automatically

### Every New Prediction (Automatic)
- Match details and predicted score
- Odds and confidence
- **LIVE system performance** (wins, ROI, profit)
- Posted automatically when generated

### Every Result (Automatic)
- Win/Loss status and actual score
- Profit/loss for that bet
- **UPDATED system stats** after result
- Posted automatically when match finishes

## Manual Updates

You can send performance updates anytime using the commands above!
